flags (max-turns/timeout enforced **driver-side**); judge = single-turn
`claude -p --model glm-5.2 --output-format json`; every commit ends

`rm -rf tests/*/.java-codebase-rag tests/*/.java-codebase-rag.{yml,hosts}`. The full bench suite is slow —
run the relevant subset during dev, the full `tests/bench/` once at the end.

`exit_reason="timeout"`; `JUDGE_CORRECT_THRESHOLD = 0.5`. `matplotlib` is an optional import (graceful).

### Task 1: Cap sentinel + capped-cell short-circuit
- **`claude_runner.run_cell`** (401-533): when the read loop sets `capped=True`, set
  `final_answer` to the sentinel `f"[BENCH_CAP: reached max-turns {spec.max_turns} without a final result]"`
  (currently `summary.final_answer` is `None` because the cap breaks before the `result` event). Keep
  `exit_reason="cap"` via `derive_exit_reason`.
- **`grade.grade_cell`** (640-696): before `GRADE_DISPATCH`, branch on
  `cell.get("exit_reason") == "cap"` → return `Grade(0.0, method=method_name, detail={"reason": "cap"})`
  with no grader/judge call. (`method_name = GRADE_DISPATCH[question.grading]`.)
- **Reuse:** `derive_exit_reason`, `GRADE_DISPATCH`.
- **Tests:** `test_run_cell_cap_sentinel_in_final_answer` (fake-claude cap fixture → final_answer is the
  sentinel, non-null), `test_grade_cell_capped_short_circuits_to_zero` (cell with exit_reason "cap" →
  Grade correctness 0.0, no judge invoked — monkeypatch judge_answer to assert not called).
- **Commit:** `fix(bench): cap sentinel in final_answer + grade_cell capped short-circuit`.

### Task 2: Driver wall-clock timeout
- **`claude_runner.run_cell`**: add param `wall_timeout_s: float | None = None`. Record `start =
  time.monotonic()` before the read loop; each iteration, if `wall_timeout_s is not None and
  time.monotonic() - start > wall_timeout_s`: set `timed_out=True`, `proc.terminate()`, `break`.
- **`derive_exit_reason(summary, capped, timed_out=False)`** (347-359): precedence
  `capped → "cap"`, `timed_out → "timeout"`, `error → "error"`, else `"done"`.
- **`run_bench.main`** CLI (302-376): add `--wall-timeout` (float, default `None`); thread through
  `run_grid` → `run_cell`. (`CellSpec` unchanged — timeout is a run_cell param, not per-cell.)
- **PREREG** Amendment 2026-07-22: record `exit_reason="timeout"` + driver wall-clock timeout.
- **Tests:** `test_run_cell_wall_timeout_terminates` (fake-claude `emit_long.sh` + `wall_timeout_s=0.1`
  → exit_reason "timeout", proc reaped), `test_exit_reason_timeout` (unit on `derive_exit_reason`).
- **Commit:** `feat(bench): driver wall-clock timeout + exit_reason="timeout"`.

### Task 3: Atomic per-cell write

- **Commit:** `fix(bench): atomic per-cell write (temp + fsync + os.replace)`.

### Task 5: `bench/report.py` (greenfield)
- **Aggregations** from `graded.jsonl` (+ `cells.jsonl` for `n_turns`/`tokens`/`context_bytes`): per
  (condition × category) and per (condition) means of correctness/steps/tokens/context; cross-service
  category by condition (C3); (model × condition) correctness delta (C6); cap/error counts; κ from the
  grade `summary.json` if present. Bucket `*_error`/`*_capped` methods as 0.0 correctness.
- **Outputs:** `report.md` (markdown tables + headline + "reproduce in 3 commands"), `results.csv`
  (one row per graded cell), `*.png` plots (matplotlib, optional — warn if absent).
- **CLI:** `report_main(argv)` with `--run-dir` (required), `--out` (default = run-dir). `python -m` not
  needed; invoke as `.venv/bin/python bench/report.py --run-dir ...`.
- **Plots:** correctness-by-category bar; correctness-vs-tokens scatter; steps-to-answer by condition;
  model deltas. **Invoke the `dataviz` skill before writing plot code** (palette + method).
- **Tests:** `test_report_aggregates_condition_means` (synthetic graded.jsonl → D mean > A mean),
  `test_report_csv_one_row_per_cell`, `test_report_handles_empty_run` (0 cells → no crash, empty report).
- **Commit:** `feat(bench): report.py — aggregate graded runs to markdown + csv + plots`.
